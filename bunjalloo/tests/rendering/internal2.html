Installation

Unzip the zip file to the root of your flash card.

Patch bunjalloo.nds for your flash card using the DLDI tool from

http://chishm.drunkencoders.com/DLDI/

Controls

Start - Enter a URL (show keyboard)
Up/Down - Scroll screen up and down
Right/Left - Scroll screen up and down by one page
R - Move forwards in the history
L - Move backwards in the history
Select - Change the position of the icons
Touch - Follow link, enter data in forms, press keys, use icons
